
### AI-Driven Adaptive Quiz

#### Overview

An adaptive quiz system that adjusts the difficulty of questions based on user performance. The AI model predicts the difficulty of the next question, providing a personalized quiz experience.

#### Features

* Difficulty level of questions (easy, medium, hard) adjusts based on previous answers.
* Real-time feedback on answer correctness.
* Automatic transition to the next question after each answer.
* Final score displayed after five rounds.

#### Requirements

* Python 3.x
* Tkinter for GUI
* Scikit-learn for machine learning

#### Installation

1. Download the project files.
2. Ensure Python is installed.
3. Install dependencies as instructed in the requirements.

#### Running the Application

1. Open the project file.
2. Run the script to begin the quiz.
3. Answer the questions displayed on the screen.

#### How It Works

* The quiz starts with a question.
* Difficulty of subsequent questions adjusts based on previous answers.
* After five rounds, the quiz ends, showing the final score.
